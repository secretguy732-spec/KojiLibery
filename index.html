Koji Livery

A premium, open-source Roblox UI library built for speed, flexibility, and visual polish. Five built-in themes, full config persistence, and a clean Lua API.

v1.0.0 MIT License Roblox Exploit UI

Quickstart #

Load Koji Livery and create your first window in just a few lines.

Lua 
-- Load the library
local Koji = loadstring(game:HttpGet('https://raw.githubusercontent.com/secretguy732-spec/KojiLibery/main/KojiLivery.lua'))()

-- Create a window
local Window = Koji:CreateWindow({
  Name            = "My Hub",
  LoadingTitle    = "My Hub",
  LoadingSubtitle = "by YourName",
  Theme           = "Midnight",
})

-- Create a tab
local MainTab = Window:CreateTab("Main", 4483362458)

-- Add a section
MainTab:CreateSection("Combat")

-- Add a button
MainTab:CreateButton({
  Name     = "Kill Aura",
  Callback = function()
    print("Activated!")
  end,
})

-- Add a toggle
MainTab:CreateToggle({
  Name     = "Speed Hack",
  Default  = false,
  Callback = function(state)
    print("Speed Hack:", state)
  end,
})

Installation #

Koji Livery is loaded via HttpGet directly from GitHub. No manual download needed.

Lua
local Koji = loadstring(game:HttpGet('https://raw.githubusercontent.com/secretguy732-spec/KojiLibery/main/KojiLivery.lua'))()
Tip: You can also copy the raw KojiLivery.lua file into your own exploit executor's file system and use loadfile or dofile for offline use.

CreateWindow #

Creates the main UI window. All tabs and elements are created from the returned Window object.

Lua
local Window = Koji:CreateWindow({
  Name             = "Koji Window",        -- Window title
  Icon             = 0,                   -- Roblox asset ID or 0 for none
  LoadingTitle     = "Koji Livery",
  LoadingSubtitle  = "by Koji",
  Theme            = "Midnight",          -- See Themes section
  HideOnClose      = false,              -- true = hide instead of destroy
  ConfigSaving = {
    Enabled    = true,
    FileName   = "MyHubConfig"
