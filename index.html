Koji Livery

A premium, open-source Roblox UI library built for speed, flexibility, and visual polish. Five built-in themes, full config persistence, and a clean Lua API.

v1.0.0 MIT License Roblox Exploit UI

Quickstart #

Load Koji Livery and create your first window in just a few lines.

Lua 
-- Load the library
local Koji = loadstring(game:HttpGet('https://raw.githubusercontent.com/koji-project/koji-livery/main/KojiLivery.lua'))()

-- Create a window
local Window = Koji:CreateWindow({
  Name            = "My Hub",
  LoadingTitle    = "My Hub",
  LoadingSubtitle = "by YourName",
  Theme           = "Midnight",
})

-- Create a tab
local MainTab = Window:CreateTab("Main", 4483362458)

-- Add a section
MainTab:CreateSection("Combat")

-- Add a button
MainTab:CreateButton({
  Name     = "Kill Aura",
  Callback = function()
    print("Activated!")
  end,
})

-- Add a toggle
MainTab:CreateToggle({
  Name     = "Speed Hack",
  Default  = false,
  Callback = function(state)
    print("Speed Hack:", state)
  end,
})

CreateWindow #

Creates the main UI window. All tabs and elements are created from the returned Window object.

Lua
local Window = Koji:CreateWindow({
  Name             = "Koji Window",        -- Window title
  Icon             = 0,                   -- Roblox asset ID or 0 for none
  LoadingTitle     = "Koji Livery",
  LoadingSubtitle  = "by Koji",
  Theme            = "Midnight",          -- See Themes section
  HideOnClose      = false,              -- true = hide instead of destroy
  ConfigSaving = {
    Enabled    = true,
    FileName   = "MyHubConfig",
  },
  KeySystem   = false,
  KeySettings = {
    Title    = "Key Required",
    Note     = "Get key from discord.gg/xyz",
    Keys     = { "my-secret-key" },
  },
})

Options

PropertyTypeDefaultDescription
Namestring"Koji Window"The title shown in the top bar.
Iconnumber|stringnilRoblox asset ID (number) or image URL.
LoadingTitlestring"Koji Livery"Large text on the loading screen.
LoadingSubtitlestring"Loading..."Subtitle below the loading title.
Themestring"Midnight"Built-in theme name. See Themes section.
HideOnClosebooleanfalseIf true, the × button hides instead of destroys the window.
ConfigSaving.EnabledbooleanfalseAutomatically saves and loads flags to file.
ConfigSaving.FileNamestring"KojiConfig"The filename used for the JSON config (no extension).
KeySystembooleanfalseIf true, shows a key input overlay before the window appears.
KeySettings.Keystable{}Array of valid key strings.
RightShift toggles window visibility globally. The ▪ minimize button collapses to the top bar only.

CreateTab #

Adds a tab to the sidebar. Returns a Tab object you use to add components.

Lua
local Tab = Window:CreateTab(
  "Players",    -- Tab name (required)
  4483362458    -- Icon asset ID (optional)
)

CreateSection #

Adds a visual divider with a label to group components within a tab.

Lua
Tab:CreateSection("Movement")

Components #

All components below are called on a Tab object.

CreateButton #

Preview
Lua
Tab:CreateButton({

CreateToggle #

Preview
Infinite Jump
Lua
local myToggle = Tab:CreateToggle({
  Name        = "Infinite Jump",
  Description = "Allows jumping infinitely.",
  Default     = false,
  Flag        = "InfiniteJump",       -- Koji.Flags["InfiniteJump"]
  Callback    = function(state)
    print("Infinite Jump:", state)
  end,
})

-- Set programmatically
myToggle:SetValue(true)
print(myToggle:GetValue())

CreateDropdown #

Lua
local teamDrop = Tab:CreateDropdown({
  Name     = "Select Team",
  Items    = { "Red", "Blue", "Green", "Yellow" },
  Default  = "Red",
  Flag     = "Team",
  Multi    = false,              -- true allows multi-select
  Callback = function(value)
    print("Selected:", value)
  end,
})

-- Refresh items dynamically
teamDrop:Refresh({ "Alpha", "Beta", "Gamma" })

CreateInput #

Lua
Tab:CreateInput({
  Name        = "Target Username",
  Placeholder = "Enter player name...",
  Default     = "",
  Numeric     = false,             -- true = number only
  Flag        = "TargetPlayer",
  Callback    = function(text, enterPressed)
    if enterPressed then
      print("Target set to:", text)
    end
  end,
})

CreateColorPicker #

Lua
Tab:CreateColorPicker({
  Name    = "ESP Color",
  Default = Color3.fromRGB(255, 0, 0),
  Flag    = "ESPColor",
  Callback = function(color)
    print("Color:", color)
  end,
})

The color picker includes a hue strip, saturation/value box, and a hex input.

CreateKeybind #

Lua
Tab:CreateKeybind({
  Name     = "Toggle ESP",
  Default  = Enum.KeyCode.X,
  Flag     = "ESPKey",
  Callback = function(key)
    print("Key pressed:", key.Name)
  end,
})

Click the key button, then press any key to rebind. The callback fires every time the bound key is pressed.

Notifications #

Shows a toast notification in the top-right corner. Independent of any window.

Lua
Koji:Notify({
  Title    = "Script Loaded",
  Content  = "Koji Livery v1.0.0 initialized.",
  Type     = "Success",   -- Info | Success | Warning | Error
  Duration = 5,           -- seconds
})

Flags #

Every component with a Flag property stores its current value in Koji.Flags, letting you read values from anywhere without holding a reference.

Lua
-- Read any flag value by name
print(Koji.Flags["WalkSpeed"])
print(Koji.Flags["InfiniteJump"])
print(Koji.Flags["ESPColor"])

Config Saving #

When ConfigSaving.Enabled is true, all flagged component values are auto-saved to a JSON file in the KojiLivery/ folder and restored on next load.

Requires: Executor must support writefile, readfile, isfile, and isfolder (most modern executors do).

Key System #

Set KeySystem = true to require users to enter a key before accessing the window.

Lua
Koji:CreateWindow({
  KeySystem   = true,
  KeySettings = {
    Title    = "Access Required",
    Note     = "Join discord.gg/myhub to get a key",
    Keys     = { "koji-2025-alpha", "vip-key-xyz" },
  },
})
